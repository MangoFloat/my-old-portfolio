Programming


Programming is one of the more enjoyable units in the course.


The unit has so far, taught us the most basic fundamentals of programming. We use a simple software, Processing to learn. So far we have been introduced to simple concepts such as sequences and variables, loops, procedures, IF statements and procedures and more.


In the first week, I made a scalable stickman (Figure 1.) as a portfolio, this stickman can be resized however big (or small) I want, I used simple operators such as *(multiply) and /(divide) to achieve this.


We were then taught about loops (for and while), and I used this for another portfolio (Figure 2.), which shows multiple monsters, increasing in size.


Figure 3 shows a simple snake/tron game. I used Booleans to add control of up, down, left, right buttons, and using it to ask whether it crashed or not.


I think programming will be one of the most useful, if not the most useful unit I will study, as it provides a lot of use application to the real world, which will definitely aid me for future.
